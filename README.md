## 开始

1. Fork [wuqifanisgood/wuqifanisgood.github.io](https://github.com/wu-kan/wu-kan.github.io/fork)到你的仓库，并在设置里开启gh-pages
2. 修改`_config.yml`为你自己的信息
3. 删除`_post/`下的博文和`public/image/`文件夹下的图片，开始写你自己的文章
4. 详细配置可以参见[这篇博文](https://wu-kan.github.io/posts/博客搭建/基于Jekyll搭建个人博客),在这里感谢wu-kan大神。
